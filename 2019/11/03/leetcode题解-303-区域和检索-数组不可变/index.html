总结

  • 傻了，老想着把计算结果存起来，到时候直接取，居然忘记了
    • \(sumrange(i,\ j)= sum[j+1] - sum[i]\)
  • 这里也用到了插入了一个虚拟 0 作为 sum 数组中的第一个元素。
    • 这个技巧可以避免在 sumrange 函数中进行额外的条件检查，对于base case, corner case等等
    • PS：尽管，我其实喜欢加上对于base case, corner case的检查，但却是实现起来有些冗余
